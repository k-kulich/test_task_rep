### Задание 1: Фермер

__Условие__

На ферме содержатся шесть разных видов животных, и каждый раз, когда фермер заходит в сарай, он видит одно случайное животное. За день фермер заходит в сарай $6$ раз.

Каково математическое ожидание количества разных видов животных, которые фермер увидит за день?


__Ответ:__ $3,99$.

__Решение__

Введем для каждого вида $i = 1, 2, \dots, 6$ индикаторную случайную величину:

$$
I_i = \begin{cases} 
    1, & \text{если вид } i \text{ был увиден хотя бы один раз за 6 посещений}, \\ 
    0, & \text{иначе}. 
\end{cases}
$$

Общее число различных видов X равно сумме этих индикаторов:

$$
X = I_1 + I_2 + \cdots + I_6
$$

По линейности математического ожидания:

$$
\mathbb{E}[X] = \sum_{i=1}^{6} \mathbb{E}[I_i] = \sum_{i=1}^{6} P(I_i = 1)
$$

Остается узнать вероятность встретить вид (все $P(I_i = 1)$ одинаковы). В каждом посещении вероятность не увидеть 1 вид равна $\frac{5}{6}$ (выбирается любой из остальных 5 видов). Так как посещения независимы, вероятность не увидеть вид 1 за 6 раз равна:

$$
P(\text{вид не увиден}) = \left(\frac{5}{6}\right)^6
$$

Следовательно, вероятность увидеть его хотя бы один раз:

$$
P(I_i = 1) = 1 - \left(\frac{5}{6}\right)^6
$$

Тогда математическое ожидание равно:

$$
\mathbb{E}[X] = 6 \left(1 - \left(\frac{5}{6}\right)^6\right) = \frac{31031}{7776} \approx 3,99.
$$

### Задание 2: Кулинарное соревнование

__Условие__

В конкурсе участвуют $80$ шеф-поваров с уникальными уровнями мастерства. В первом этапе судьи случайным образом распределяют их по парам (в любом состязании двух шефов выигрывает тот, у кого выше уровень мастерства). На втором этапе шефы снова случайно образуют пары для финального раунда (пары могут повториться). Победную награду получают те, кто выиграл в обоих этапах.

Каково математическое ожидание числа победителей?

__Ответ:__ $26,8$.

__Решение__

Обозначим $n = 80$ – количество шеф-поваров. У каждого повара свой уникальный уровень мастерства. Пронумеруем поваров в порядке возрастания мастерства: $1$ – самый слабый, $80$ – самый сильный.  

Для повара с номером $k$ ($1 \le k \le 80$) количество поваров, имеющих меньший уровень, равно $k-1$.  

В первом этапе повара случайным образом разбиваются на пары. Для фиксированного повара его соперник выбирается равновероятно среди всех $79$ остальных. Поэтому вероятность того, что он выиграет в своей паре, равна

$$
p_k = \frac{k-1}{79}
$$

Аналогично, во втором этапе пары образуются заново независимо от первого, и вероятность выиграть для того же повара снова равна $p_k$. Поскольку выборы соперников в двух этапах независимы, вероятность выиграть в обоих этапах равна (если событие $A$ - победа повара $k$).

$$
P(A) = p_k^2 = \left(\frac{k-1}{79}\right)^2
$$

Пусть $X$ – общее число победителей (поваров, выигравших в обоих этапах). Введем индикаторы $I_k$ для каждого повара: $I_k = 1$, если повар $k$ выиграл в обоих этапах, и $0$ иначе. Тогда $X = \sum_{k=1}^{80} I_k$. По линейности математического ожидания

$$
\mathbb{E}[X] = \sum_{k=1}^{80} \mathbb{E}[I_k] = \sum_{k=1}^{80} \left(\frac{k-1}{79}\right)^2
$$

Что представляет из себя сумму квадратов чисел от $0$ до $79$, которая вычисляется как:

$$
\sum_{k=1}^{80} \left(\frac{k-1}{79}\right)^2 = \frac{1}{79^2} \sum_{k=1}^{80} (k-1)^2 = \frac{n(n+1)(2n+1)}{6}
$$

При $n = 79$ получаем:

$$
\frac{n(n+1)(2n+1)}{6} = \frac{79 \cdot 80 \cdot 159}{6} = \frac{1}{79^2} \cdot \frac{79 \cdot 80 \cdot 159}{6} = \frac{80 \cdot 159}{6 \cdot 79} = \frac{80 \cdot 159}{474}
$$

Таким образом, в результате вычислений получаем:

$$
\mathbb{E}[X] = \frac{2120}{79} = 26\frac{66}{79} \approx 26,8.
$$

### Задание 3: Одинокая дорога

__Условие__

На пустынном шоссе вероятность появления автомобиля за $30$-минутный период составляет $0,95$.

Какова вероятность его появления за $10$ минут? А за $27$ минут?

__Ответ:__ $63,2; 93,3$.

__Решение__

Предполагаем, что поток автомобилей является простейшим (пуассоновским) с постоянной интенсивностью $\lambda$. Вероятность появления хотя бы одного автомобиля за время $t$ равна:

$$
P(t) = 1 - e^{-\lambda t}
$$

Из условия $P(30) = 0,95$ находим $\lambda$:

$$
1 - e^{-30\lambda} = 0,95 \quad\Rightarrow\quad e^{-30\lambda} = 0,05 \quad\Rightarrow\quad \lambda = -\frac{\ln 0,05}{30}
$$

$\ln 0,05 \approx -2,995732$, поэтому $\lambda \approx 0,0998577$.

Тогда:

- за $10$ минут: $P(10) = 1 - e^{-10\lambda} = 1 - e^{-0,998577} \approx 1 - 0,3685 = 0,6315 \approx 63,2%$.

- за $27$ минут: $P(27) = 1 - e^{-27\lambda} = 1 - e^{-2,69616} \approx 1 - 0,0674 = 0,9326 \approx 93,3%$.

### Задание 1: Изоморфизмы

Мое решение с примерами использования приведено в файле `task1.py`, код самой функции:

```python
def is_isomorphic(s: str, t: str) -> bool:
    if len(s) != len(t):
        return False
    map_s_to_t = {}
    map_t_to_s = {}
    for cs, ct in zip(s, t):
        if ct != map_s_to_t.get(cs, ct):
            return False
        else:
            map_s_to_t[cs] = ct
        
        if cs != map_t_to_s.get(ct, cs):
            return False
        else:
            map_t_to_s[ct] = cs
    return True
```

__Оценка алгоритма:__

- $O(n)$ по времени, так как он необходимо пройтись по всей длине строки (только строки одинаковой длины могут быть изоморфами), чтобы проверить изоморфность.

- $O(k)$ по памяти, где $k \le n$ - число уникальных символов строки.

def missing_number(nums: list) -> int:
    n = len(nums) + 1                     
    total_expected = n * (n + 1) // 2

### Задание 3: Факторизация

Мое решение с примерами использования приведено в файле `task3.py`, код самой функции:

```python
def prime_factors(num: int) -> list:
    factors = []
    while num % 2 == 0:
        factors.append(2)
        num //= 2
    d = 3
    while d * d <= num:
        while num % d == 0:
            factors.append(d)
            num //= d
        d += 2
    if num > 1:
        factors.append(num)
    return factors
```

__Оценка алгоритма:__

- $O(\sqrt{n})$ по времени, так как необходимо обойти все числа до $\sqrt{n}$ (оптимальный обход при поиске делителей числа).

- $O(\log_2 {n})$ по памяти, так как необходимо хранить делители, которых не больше $\log_2 {n}$.

Необходимо было вывести ID клиентов, которые за последний месяц по всем своим счетам совершили покупок меньше, чем на $5000$ рублей.

__Решение:__

```SQL
SELECT 
    a.client_id
FROM account a
LEFT JOIN transaction t 
    ON a.id = t.account_id
    AND t.type = 'buy'
    AND t.transaction_date >= CURRENT_DATE - INTERVAL '1 month'
GROUP BY a.client_id
HAVING COALESCE(SUM(t.amount), 0) < 5000;
```

### Задание 2: Ручной счёт ROC_AUC

__Решение:__

- Ранжируем по возрастанию вероятности. Получим число положительных примеров (порог пройден) $P=7$, отрицательных $N=8$.

- Считаем сумму рангов положительных примеров: $S = 4+8+9+10+11+13+15=70$.

### Задание 3: Ручной счет корреляции

__Ответ:__ –0,85 (округление до сотых).

__Вывод:__ Обнаружена сильная отрицательная линейная корреляция между количеством выпитого кофе и экзаменационным баллом. Однако __утверждать о причинно-следственной связи нельзя__. Корреляция может быть вызвана скрытыми факторами (например, стрессом, недостатком сна, которые одновременно увеличивают потребление кофе и снижают результативность).

__Решение:__

| Чашки кофе (X) | Балл (Y) |
|----------------|----------|
| 1              | 85       |
| 1              | 88       |
| 2              | 79       |
| 2              | 81       |
| 2              | 84       |
| 2              | 65       |
| 3              | 67       |
| 3              | 58       |
| 3              | 76       |
| 4              | 49       |

__Вычисление коэффициента корреляции Пирсона $r$:__  
$n = 10$  
$\Sigma X = 23$, $\Sigma Y = 732$  
$\Sigma X^2 = 61$, $\Sigma Y^2 = 55082$  
$\Sigma XY = 1590$

$$
r = \frac{n\Sigma XY - (\Sigma X)(\Sigma Y)}{\sqrt{[n\Sigma X^2 - (\Sigma X)^2][n\Sigma Y^2 - (\Sigma Y)^2]}}
$$
$$
r = \frac{10 \cdot 1590 - 23 \cdot 732}{\sqrt{[10 \cdot 61 - 23^2][10 \cdot 55082 - 732^2]}}
= \frac{-936}{\sqrt{81 \cdot 14996}}  \approx -0.8492
$$
